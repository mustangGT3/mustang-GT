- Primary keyword in position 1-3
- Brand name "Mustang GT" preferably at start
- Modifiers like year (2024, 2025), spec (specs, price), or action (buy, for sale)

### Meta Description Keywords (155-160 chars)
- Include primary keyword naturally
- Add secondary keyword if space allows
- End with CTA (Check our pricing, Explore models, Learn more)

### Body Content Keywords (Natural Integration)
- Heading (H1, H2) contains primary keyword
- First 100 words include keyword at least once
- Related keywords in subheadings (H3, H4)
- Contextual internal links with keyword anchor text
